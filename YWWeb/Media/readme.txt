
文件说明:
	1 NVR2_Web_plugin.exe:这个安装包，包含IE、chrome、firefox插件，限于windows电脑上使用，chrome只支持45以下版本(不包括45)
	2 SurveillanceClient：苹果safari插件，限于MAC电脑上的safari、不包括windows safari、手机或平板safari
	
使用说明：
	1 使用前请先安装插件
	2 windows插件安装的目录在C:\Users\xxxxxx\AppData\Roaming\WebPlugins\NVR2,其中xxxxxx代表电脑账户
	3 苹果插件安装在:/Library/Internet Plug-Ins/
	4 插件安装完成后用浏览器打开login.html，在出现的登录页面里输入对应的设备信息，然后点击登录.比如:
	   IP Address:CHAA1504581582285   (这里填写NVR真实的ID)
	   Username: admin				  (NVR 设备的用户名)
	   Password:                      (NVR 设备的登录密码)
	   Client Port:55552      		  (NVR出厂默认是9000)
	   其它选项默认即可.
